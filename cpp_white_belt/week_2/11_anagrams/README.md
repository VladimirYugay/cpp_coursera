#### Anagrams ####

Check whether words are anagrams 


##### Input Format #####
N - number of pairs 
word_a, word_b on each line 

##### Output Format#####
For each pair print "YES" or "NO" 

##### Example #####
###### Input ######
```objectivec
3
eat tea
find search
master stream
```

###### Output ######
```objectivec
YES
NO
YES
```
